7.1 Creating Contexts

— Function: gpgme_error_t gpgme_new (gpgme_ctx_t *ctx)

The function gpgme_new creates a new gpgme_ctx_t object and returns a handle for it in ctx.

The function returns the error code GPG_ERR_NO_ERROR if the context was successfully created, GPG_ERR_INV_VALUE if ctx is not a valid pointer, and GPG_ERR_ENOMEM if not enough memory is available.
