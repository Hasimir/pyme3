PyMe

PyMe is a python interface to GPGME library. PyMe is a GPGME + Python + SWIG (just like m2crypto is an OpenSSl + Python + SWIG) combination which means that most of the functions and types are converted from C into Python automatically and therefore to be able to use PyMe you need to be familiar with GPGME. PyMe files and CVS can be found on the PyMe SourceForge project page. GPGME files and CVS can be found on the GPGME FreshMeat project page. SWIG download files can be found on the SWIG SourceForge project page.
