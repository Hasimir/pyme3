PyMe

PyMe is a python interface to GPGME library. PyMe is a GPGME + Python + SWIG (just like m2crypto is an OpenSSL + Python + SWIG) combination which means that most of the functions and types are converted from C into Python automatically and therefore to be able to use PyMe you need to be familiar with GPGME.

PyMe release files and CVS can be found on the PyMe SourceForge project page.
GPGME release files and CVS can be found on the GPGME FreshMeat project page.
SWIG download files can be found on the SWIG SourceForge project page.

Previouse version of PyMe 0.5.1 which works with GPGME 0.3.15 can be found on John Goerzen's PyMe page: http://quux.org/devel/pyme/
